What is a Framework


• Framework is a set of rules and ideas that provide structure to a complex and tough situation.
• Framework structure is rigid but provides flexibility to business objectives and activities
• Framework depends upon the business requirements
• Framework provides a consistent system of controls to which IT department can adhere providing a consistent approach to auditor for conducting audits
• A more governing and descriptive type of framework dictate a control objective that IT organisation should ensure system security


Most Frameworks are referred to as Standards. Some Standards however wouldn’t be called frameworks.
