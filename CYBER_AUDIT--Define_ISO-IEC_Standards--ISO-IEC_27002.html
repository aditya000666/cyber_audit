ISO/IEC 27002


• Provides suitable information security controls within the ISMS
• Merely a code of practice or guideline rather than certification standard
• Organisation are free to select and put in place other controls as they see fit
• 18 sections make up of ISO/IEC 27002
• Framework applies to the key sections which are summarised as list
◇ Information Security Policies
◇ Organisation of Information Security
◇ Human Resource Security
◇ Asset management
◇ Access Control
◇ Cryptography
◇ Physical and Environmental Security
◇ Operations Security
◇ Communication Security
◇ Systems Acquisition, Development and Maintenance
◇ Supplier Relationships
◇ Information Security Incident Management
◇ Information Security Aspects of Business Continuity Management
◇ Compliance
