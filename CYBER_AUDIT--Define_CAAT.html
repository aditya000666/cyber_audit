Define CAAT


Computer Assisted Audit Technique



• Another type of software technique used in IT audits
• Can be used by both IT or Financial auditors in variety of ways to evaluate the integrity of applications, determine compliance with procedures and continuously monitor processing results
• Common CAAT like ACL and Interactive Data Extraction Analysis (IDEA) can be used to select sample, analyse characteristics of data file, identify trends in data and evaluate data integrity
• Microsoft Access and Microsoft Excel can also be used to analyse data, creating graphs and performing regression and trend analysis
