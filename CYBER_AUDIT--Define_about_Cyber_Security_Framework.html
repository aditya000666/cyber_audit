Define about Cyber Security Framework


• NIST released the Cyber Security Framework in 2014
• Improve the Critical Infrastructure Cyber Security
• Collaboration of Government and Private-Sector participants
• Provide voluntary structure for reducing risks to critical infrastructure

• Cyber Security Framework is made up of 3 components
◇ Framework Core
◇ Framework Profile
◇ Framework Implementation Tiers
• Framework Core is matrix of activities and associated references

• Different categories across Framework Core
◇ Identify
◇ Protect
◇ Detect
◇ Respond
◇ Recover
• The Framework Profile provides the primary mechanism to improve the security posture by comparing the existing state or profile with ideal end state or target profile
