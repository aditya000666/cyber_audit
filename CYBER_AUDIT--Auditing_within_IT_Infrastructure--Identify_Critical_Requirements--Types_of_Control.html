Types of Control


• Management Control
• Governed by the management
• Contains the following
• Security policy
• Security management
• Risk management
• Assurance
• Operational Control
• Implemented by people rather than system
• Interrelated to both management and technical control
• Contains the following
• Incident Response handling
• Awareness, training and education
• Physical and Environment security
• Personal and User issues
• Technical Control
• Include controls performed by IT
• Include the following
• Identification and Authorisation
• Logical Access control
• Cryptography
